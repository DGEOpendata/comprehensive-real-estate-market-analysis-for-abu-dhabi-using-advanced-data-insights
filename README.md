# Comprehensive Real Estate Market Analysis Tool

## Overview
This tool provides a comprehensive analysis of the Abu Dhabi real estate market using an integrated dataset. The platform allows users to visualize property prices, rental trends, and transaction volumes across different neighborhoods and time periods.

## Features
- **Neighborhood Price Analysis**: Visualize average property prices across different neighborhoods.
- **Rental Yield Trends**: Analyze changes in rental yields over the years.
- **Transaction Volume Visualization**: Track total transaction volumes over time.

## Getting Started
1. **Prerequisites**: Ensure Python is installed on your system along with `pandas` and `matplotlib` libraries.
   bash
   pip install pandas matplotlib
   

2. **Dataset**: Download the `Abu_Dhabi_Real_Estate_Market_Insights.csv` dataset and place it in your working directory.

3. **Run the Script**: Execute the provided Python script to load data and generate visualizations.

## Usage
- The script reads the dataset, processes the data, and generates visualizations.
- Modify the script to filter data or customize plots according to your needs.
